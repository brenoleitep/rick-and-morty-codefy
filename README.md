## Como rodar o projeto

1.  Clone o repositório na sua maquina;
2.  Abra o seu terminal
3.  Instale as dependencias com Yarn
4.  Yarn run build para buildar o projeto
5.  Yarn run dev
6.  Abrir localhost na porta indicada pelo terminal
